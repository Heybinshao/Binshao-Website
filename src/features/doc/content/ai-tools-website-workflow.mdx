## AI工具流：小网站速成记

年前做了一个小网站，专门用来展示各大 APP 的年度报告。今天来复盘一下整个制作流程，顺便分享下我用到的工具链。

### 从 YouMind 开始规划

项目的起点是在 YouMind 里完成的。我先把想法和大致的页面结构梳理清楚，然后直接让 AI 帮我生成代码。

### 用 Trae 微调细节

从 YouMind 导出代码后，我把项目放到了 Trae 里继续开发。

### 图片压缩：TinyPNG4Mac

做展示类网站，图片往往是大头。我用的是 TinyPNG4Mac 这个开源工具来批量压缩图片。

### 部署到 Cloudflare

最后一步是把网站部署上线。我选择了 Cloudflare Pages，原因很简单：免费、速度快、配置简单。
