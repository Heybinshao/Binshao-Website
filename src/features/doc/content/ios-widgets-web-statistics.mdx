## iOS 小组件查看网站访问数据

想随时在 iPhone 主屏查看自己网站的访问数据？利用 Scriptable 工具，只需几十行代码，就能实现一个美观实用的小组件。

### 工具准备

- Scriptable：通过使用 JavaScript 来创建小组件的应用
- 51.LA：国内网站访问统计平台
